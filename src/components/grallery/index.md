## API

### Grallery
| 属性 | 说明 | 类型 | 默认值 |
| :---- | :---- | :---- | :------ |
| gap | 缝隙 | number | - |
| itemWidth | 缝隙 | number | - |
| showQuantity | 显示item的数量 | number | 3 |
| itemClick | item点击事件(参数为item的索引) | number | - |
